Sarvam AI assistant for NVDA

An accessible NVDA add-on that brings the Sarvam AI platform to blind and low-vision users through a fully keyboard-driven, screen-reader-friendly interface.

Features

  • Text to Speech – synthesise Indian-language speech from a selection, the clipboard, typed text or a text file; play, pause, resume, stop and save the audio.
  • Speech to Text – transcribe an audio file or your microphone, optionally translating to English; copy or save the transcript.
  • Translation & Transliteration – convert text between English and 10 Indian languages, with selectable styles.
  • Language Detection – identify the language and script of any text.
  • AI Assistant – chat, summarise text and understand documents with Sarvam's language models.
  • OCR – recognise text from PDFs and images with Sarvam Document OCR (23 languages) or the offline Windows OCR engine, then translate or summarise it with Sarvam.

About OCR: The primary engine is Sarvam's Document Digitization API, which recognises 23 languages from a PDF or image. Images and screen/clipboard captures are automatically wrapped into a one-page PDF before being sent. A second, fully offline Windows OCR engine is also available for instant screen and object recognition.

Getting started

  1. Open NVDA menu → Tools → Sarvam AI → Settings…
  2. Paste your Sarvam API key (from https://dashboard.sarvam.ai).
  3. Press Validate key / Test connection.
  4. Set your default voice, language and folders, then press OK.

Menu

All features are under Tools → Sarvam AI: Text to Speech, Speech to Text, Translate, Transliterate, Detect language, OCR, AI Assistant / Summarise, Settings, Logs, Check for updates, Help and About.

Keyboard shortcuts

No gestures are assigned by default. Add your own under NVDA → Preferences → Input Gestures → Sarvam AI.

Text to Speech

Type or load text, choose a voice and language, then press Speak. Use Save audio… to export a WAV file. Playback can be paused, resumed and stopped.

Speech to Text

Choose the audio language (or leave it on automatic), then either Transcribe audio file… or Start recording to capture your microphone. Tick Translate to English to receive an English transcript. Copy or save the result.

Translate, Transliterate, Detect language

Pick the operation, enter or load text, choose source and target languages, then press Go.

AI Assistant

Choose a task – chat, summarise or understand a document – enter your text or question and press Send.

OCR

Choose an engine: Sarvam Document OCR (online, 23 languages, best for PDFs and photos) or Windows OCR (offline, instant, best for reading the screen). Recognise a PDF/image file, a clipboard image, or the current screen/object. Recognised text can be copied, saved, translated or summarised.

Troubleshooting

SymptomCause / fix
"No Sarvam API key set"Enter your key in Settings.
"Authentication failed"The key is wrong; re-check it.
"No credits available"Your account needs credits – add them in the Sarvam dashboard.
"Windows OCR is not available"Install a Windows OCR language pack in Windows Settings.
Nothing happens on the networkCheck your connection, proxy and network timeout in Settings; open Logs for details.

Privacy

Your API key is stored obfuscated in NVDA's configuration. Text, audio and images you process are sent to Sarvam for processing; OCR runs locally in Windows.
